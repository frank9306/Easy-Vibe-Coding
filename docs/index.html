安装级别

从你当前需要的能力开始。

级别越高，覆盖的工程场景越完整。随时可以重新选择，不必一次装满。

Lite

刚开始使用 Vibe Coding，建立编码、开发流程和 Git 安全底线。

--level lite

Standard

推荐

独立完成真实项目，补齐测试、安全、审查、上下文和验证能力。

--level standard

Pro

维护长期工程，加入架构决策、设计模式、性能和代码库理解能力。

--level pro

Team

面向多人或多 Agent 协作，增加安全扫描、评估和工程治理。

--level team

工作方式

每一步都清楚，
每个改动都可追溯。

  1. 01

    检测项目

    识别 Agent、项目类型和目标目录。

  2. 02

    选择能力

    应用预设级别，或精确选择单个组件。

  3. 03

    审查计划

    写入前显示来源版本和全部目标文件。

  4. 04

    安全落盘

    记录文件与 SHA-256，卸载时保护用户修改。
